# Geese Minecraft Autopacker

## PROPOSAL
- description
    - download pack
- flags
    - dot-minecraft (-M)
        - put everything by type in a .minecraft folder arrangement
    - force (-F)
        - sync even if the file exists
    - auto (-A)
        - also sync required components, such as dependencies
    - test (-T)
        - test run only, dont write file, always force
    - verbose (-V)
        - print detailed error message
    - elaborate (-E)
        - performs extra operations to ensure no conflicts happen and/or provide additional information on errors
    - file (-f)
        - input data file
    - directory (-d)
        - output directory
    - nyaa (-N)
        - nyaaaa :3
